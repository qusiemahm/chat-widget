‚úÖ Steps to Enable CDN Deployment

For jsDelivr/unpkg:
  1. Commit the dist/ folder to git
  2. Create and push a version tag (e.g., git tag v1.0.0)
  3. Push to a public GitHub repository
  4. Wait 12-24 hours for jsDelivr to index it
  5. Test the CDN URL

üîí Security Best Practices

  • API Keys: Use widget-specific API keys with limited scopes
  • Rate Limiting: Implement on backend to prevent abuse
  • CORS: Configure allowed origins (don't use wildcard in production)
  • SRI: Use Subresource Integrity hashes for CDN files
<!-- With SRI hash for security -->
<script 
  src="https://cdn.jsdelivr.net/gh/username/chatbot-widget@1.0.0/dist/TharwahChat-V1.js"
  integrity="sha384-YOUR_HASH_HERE"
  crossorigin="anonymous"
></script>
