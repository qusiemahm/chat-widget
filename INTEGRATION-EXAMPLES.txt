📋 QUICK INTEGRATION (3 Steps)

1. Copy file:        tharwah-widgets/dist/TharwahChat-V1.js
2. Add config:       <script>window.tharwahChatConfig = {...}</script>
3. Load widget:      <script src="./TharwahChat-V1.js"></script>

🚀 PRODUCTION CHECKLIST

For production websites:

1. Update API endpoint:
   apiEndpoint: 'https://your-production-api.com/api'

2. Use production API key:
   apiKey: 'prod_your_production_key'

3. Disable debug:
   debug: false

4. Configure CORS in backend:
   CORS_ALLOWED_ORIGINS=https://yourwebsite.com

📁 FILE LOCATION

Put TharwahChat-V1.js anywhere on your site:

your-website/
├── js/TharwahChat-V1.js               → <script src="js/...">
├── assets/TharwahChat-V1.js           → <script src="assets/...">
├── public/widgets/TharwahChat-V1.js   → <script src="public/widgets/...">
└── cdn/TharwahChat-V1.js              → <script src="cdn/...">

Just update the src path to match!
